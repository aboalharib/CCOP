# 1 Purpose

This document defines the **data model and telemetry specification** for the Climate Change Operation Platform (CCOP).

The specification describes how environmental telemetry is structured, encoded, transmitted, stored, and processed within the system.

---

# 2 Scope

This specification applies to telemetry generated by:

- TerraNode monitoring stations  
- TerraEdge monitoring stations  
- TerraAir monitoring stations  

The specification defines the structure of environmental data transmitted to the CCOP Climate Intelligence Platform.

# 4 Data Model Overview

The CCOP data model represents environmental observations collected by distributed monitoring stations.

Each telemetry message represents a **timestamped environmental observation event**.

Telemetry records include:

- station identification  
- monitoring role  
- timestamp  
- geospatial location  
- environmental measurements  
- system health status  

---

# 5 Monitoring Station Identification

Each monitoring station has a unique identifier.

Format:

CCOP-[TYPE]-[NUMBER]


Examples:


CCOP-TN-001
CCOP-TE-015
CCOP-TA-007


| Code | Station |
|------|--------|
| TN | TerraNode |
| TE | TerraEdge |
| TA | TerraAir |

---

# 6 Monitoring Role Field

Telemetry messages include a field identifying the monitoring role.

| Value | Meaning |
|------|--------|
| reference | TerraNode |
| distributed | TerraEdge |
| portable | TerraAir |

This field reflects the **Environmental Monitoring Pyramid model**.

---

# 7 Timestamp Model

Telemetry timestamps follow **ISO 8601 format**.

Example:


2026-05-01T14:22:30Z


This timestamp represents the time the observation was recorded.

---

# 8 Geospatial Metadata

Monitoring stations provide geographic location metadata.

| Field | Description |
|------|-------------|
| latitude | decimal degrees |
| longitude | decimal degrees |

Example:


latitude: 25.2854
longitude: 51.5310


Geospatial data enables mapping and spatial analysis.

# 10 Telemetry Message Structure

Telemetry messages include the following fields.

| Field | Description |
|------|-------------|
| station_id | unique station identifier |
| device_type | TerraNode / TerraEdge / TerraAir |
| monitoring_role | reference / distributed / portable |
| timestamp | observation time |
| location | geographic coordinates |
| measurements | environmental measurements |
| system_status | device health status |

---
