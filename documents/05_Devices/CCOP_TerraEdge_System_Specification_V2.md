# 1 Purpose

This document defines the hardware, software, and operational specification for the TerraEdge environmental monitoring station within the CCOP monitoring network.

TerraEdge is designed as a distributed environmental monitoring node providing localized sensing coverage within the CCOP environmental monitoring architecture.

---

# 2 Official Definition

TerraEdge is a distributed environmental monitoring edge device within the CCOP (Climate Change Operation Platform) designed to measure air quality and environmental conditions and transmit telemetry to the **CCOP Climate Intelligence Platform (CIP)** for analysis and monitoring.

---

# 3 Role in CCOP Monitoring Architecture

The CCOP monitoring system follows a **three-tier architecture**.

| Tier | System | Role |
|-----|------|------|
| Tier 1 | Reference Stations | calibration and baseline measurements |
| Tier 2 | TerraNode | regional environmental monitoring |
| Tier 3 | TerraEdge / TerraAir | localized sensing coverage |

TerraEdge expands monitoring density and supports site-level environmental analysis.

---

TerraEdge uses a subset of the TerraNode sensor architecture.

Typical sensors include:

| Sensor | Parameter |
|---|---|
SEN0460 | PM1 / PM2.5 / PM10 |
CO sensor | Carbon Monoxide |
O₃ sensor | Ozone |
NO₂ sensor | Nitrogen Dioxide |
SO₂ sensor | Sulfur Dioxide |

HF sensor | Hydrogen Fluoride |
H₂S sensor | Hydrogen Sulfide |
SEN0636 | UV Index |

# 11 Sensor Lifecycle Monitoring

TerraEdge implements automatic sensor lifecycle monitoring to support preventive maintenance.

The lifecycle architecture is defined in:

**CCOP_Sensor_Lifecycle_Management_Architecture_V1.md**

Each applicable sensor maintains lifecycle metadata including:

- installation date
- expected lifespan
- sensor serial number
- replacement history

The system automatically calculates:

| Metric | Description |
|---|---|
days_used | operational days since installation |
days_remaining | estimated remaining lifespan |
life_percent_remaining | remaining operational life |
lifecycle_status | operational state |

Lifecycle status categories:

| Status | Condition |
|---|---|
OK | more than 180 days remaining |
WARNING | 90–180 days remaining |
CRITICAL | less than 90 days remaining |
EXPIRED | sensor lifespan exceeded |
