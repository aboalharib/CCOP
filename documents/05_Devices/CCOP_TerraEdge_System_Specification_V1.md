# 1 Purpose

This document defines the hardware specification for the **TerraEdge monitoring station** used in the CCOP monitoring network.

TerraEdge serves as a **distributed monitoring station** within the CCOP Environmental Monitoring Pyramid.

---

# 2 Role in the Monitoring Architecture

The monitoring pyramid consists of three station classes.

| Tier | Station | Role |
|---|---|---|
| Tier 1 | TerraNode | Reference monitoring station |
| Tier 2 | TerraEdge | Distributed monitoring station |
| Tier 3 | TerraAir | Portable monitoring station |

TerraEdge stations expand monitoring coverage across large geographic areas.

---

# 3 Functional Objectives

The TerraEdge station must support:

- continuous environmental monitoring
- air quality observation
- telemetry transmission to the CCOP Climate Intelligence Platform (CIP)
- comparison with TerraNode reference stations
- support for environmental data governance

---

The TerraEdge architecture includes:


Environmental Sensors

Power Management System

Telemetry Transmission

# 5 Sensor Suite

TerraEdge includes environmental sensors but typically **excludes meteorological sensors**.

| Sensor Type | Measurement |
|---|---|
Particulate Sensor | PM1 / PM2.5 / PM10 |
NDIR Sensor | CO₂ |
Electrochemical Sensors | NO₂ / SO₂ / O₃ |
VOC Sensor | Volatile Organic Compounds |
Temperature Sensor | Ambient temperature |
Humidity Sensor | Relative humidity |
Pressure Sensor | Atmospheric pressure |
UV Sensor | UV index |

---

# 6 Embedded Processing Unit

The embedded processing unit is responsible for:

- collecting sensor measurements
- formatting telemetry messages
- generating timestamps
- monitoring device health
- communicating with the CCOP CIP backend platform

Possible hardware platforms include:

- microcontrollers
- embedded Linux systems
- industrial IoT boards

---

# 7 Communication Module

Telemetry transmission may use:

- cellular communication
- wireless IP communication
- Ethernet connectivity where available

All telemetry is transmitted to the **CCOP Climate Intelligence Platform (CIP)**.

---

# 8 Power System

TerraEdge stations may use:

- solar power systems
- battery storage
- external power where available

The power system must support long-term field deployment.

---

# 9 Environmental Tolerance

TerraEdge stations must operate in harsh environments.

Environmental requirements include:

- high temperature tolerance
- humidity resistance
- dust protection
- weatherproof enclosure

Typical enclosure rating:


IP65 or higher


---

# 10 Telemetry Output

Telemetry messages generated by TerraEdge follow the CCOP telemetry schema.

Telemetry includes:

- station identifier
- timestamp
- environmental measurements
- calibration metadata
- governance metadata

---

# 11 Calibration Relationship

TerraEdge stations may compare measurements with nearby TerraNode stations to detect sensor drift.

Calibration workflows are defined in the **CCOP Sensor Calibration Framework**.
