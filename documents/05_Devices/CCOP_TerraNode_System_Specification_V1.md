# 1 Purpose

This document defines the hardware specification for the **TerraNode monitoring station** used in the CCOP monitoring network.

TerraNode serves as the **reference monitoring station** within the CCOP Environmental Monitoring Pyramid.

---

# 2 Role in the Monitoring Architecture

The monitoring pyramid consists of three station classes:

| Tier | Station | Role |
|---|---|---|
| Tier 1 | TerraNode | Reference monitoring station |
| Tier 2 | TerraEdge | Distributed monitoring station |
| Tier 3 | TerraAir | Portable monitoring station |

TerraNode provides the most complete environmental observations and supports calibration reference functions.

---

# 3 Functional Objectives

The TerraNode station must support:

- continuous environmental monitoring
- meteorological observation
- telemetry transmission to the CCOP CIP platform
- reference measurements for calibration
- support for environmental data governance

---

|---|---|

NDIR Sensor | CO₂ |

VOC Sensor | Volatile Organic Compounds |
Temperature Sensor | Ambient temperature |
Humidity Sensor | Relative humidity |
Pressure Sensor | Atmospheric pressure |
UV Sensor | UV index |

# 7 Embedded Processing Unit

The embedded processing unit is responsible for:

- sensor data acquisition
- telemetry formatting
- timestamp generation
- device health monitoring
- communication with the backend platform

Possible hardware platforms include:

- industrial microcontrollers
- embedded Linux systems

---

# 8 Communication Module

Telemetry transmission may use:

- cellular communication
- wireless IP communication
- Ethernet connectivity where available

The communication module transmits telemetry to the **CCOP Climate Intelligence Platform (CIP)**.

---

# 9 Power System

The TerraNode station may use:

- solar power systems
- battery storage
- external power where available

Power management ensures continuous operation.

---

# 10 Environmental Tolerance

The TerraNode station must operate in harsh environments.

Environmental requirements may include:

- high temperature tolerance
- humidity resistance
- dust protection
- weatherproof enclosure

Typical enclosure rating:


IP65 or higher

# 11 Telemetry Output

Telemetry messages generated by TerraNode follow the CCOP telemetry schema.

Telemetry includes:

- station identifier
- timestamp
- environmental measurements
- calibration metadata
- governance metadata

---

# 12 Calibration Role

TerraNode stations act as **reference calibration stations**.

Distributed monitoring stations such as TerraEdge may compare measurements against TerraNode data to detect drift.
